# Girlie vs BoyZ Origins Addon
A small **Origins** addon for **Minecraft 1.20.4** that adds two fun and intentionally uhh balanced origins. :)
> This addon is meant for fun and doesn't mean to offend or disrespect anyone
## Requirements
- Minecraft 1.20.4
- Fabric
- Origins Mod

  --------
# Origins (Descriptions)
  ## Girlie Power 💎
  *This will make you incredibly weak at all times especially mining or fighting.*
  ### Abilities
  - Become invisible while crouching
  - Gain Weakness III while mining blocks (you are so welcome)
